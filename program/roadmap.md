# EPOS Master Roadmap

## Purpose

This document serves as the master program roadmap for EPOS.

It provides a high-level view of the program, releases, phases, timelines, and major milestones.

Detailed execution tracking, release progress, risks, and daily updates are maintained in their respective program documents.

# Roadmap Governance

The master roadmap is a planning artifact and should only be updated when one or more of the following occur:

- Program scope changes
- A release is added, removed, or redefined
- Phase sequencing changes
- Major milestones change
- Program timelines are formally revised

Execution progress, release status, risks, dependencies, and daily engineering activities are maintained in the corresponding release plans, governance documents, and daily engineering logs.
